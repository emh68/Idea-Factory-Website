Programming - Python
(Beginner Age 12-16)

This class is taught using the Python programming language and is for those who have little to no programming experience. The course will cover the basics of Python programming up to, but not including object oriented programming (OOP). This is not a Scratch course (although Scratch may be used from time to time if needed to explain some concepts), students will actually be writing lines of code in the IDE. Here are some of the things covered in the class:

  • setup, syntax and comments
  • variable, data types and numbers
  • strings, booleans and operators
  • Lists
  • Dictionaries
  • If/Else
  • Loops (for/while)
  • Functions
