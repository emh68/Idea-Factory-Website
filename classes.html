Programming - Python Course Details

Course Dates

Month Dates
November 12th, 19th
December 3rd, 10th, 17th
January 7th, 14th, 21st, 28th
February 4th, 11th
Location: Conference Room @ 343 E 4th N, Rexburg, ID 83440
Class Time: Tuesdays 4:00pm-6:00pm
Age Group: 12 - 16
Instructor: Eli Hansen
Cost: $200/month

Course Info:

This course is taught using the Python programming language and is for those who have little to no programming experience. The course will cover the basics of Python programming up to, but not including object-oriented programming (OOP). This is not a Scratch course (although Scratch may be used from time to time to explain some concepts), students will actually be writing lines of code in the IDE. Here are some of the topics covered:

  • Setup, syntax, and comments
  • Variables, data types, and numbers
  • Strings, booleans, and operators
  • Lists
  • Dictionaries
  • If/Else
  • Loops (for/while)
  • Functions
